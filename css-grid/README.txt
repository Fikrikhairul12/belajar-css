CSS Grid
CSS Grid adlh untk mendefinisikn sistem layout berbntuk grid dlm 2 dimensi (baris & kolom).
Grid bkn pengganti dari Flexbox.


istilah istilah:
- grid container
- grid item
- grid line
- grid cell
- grid area
- grid track
- grid gap

Grid Container:
"elemen pembungkus grid, didefinisikan dgn menulis: 
display: grid;"

Grid Item:
"elemen2 yg berada (1 level) di dlm grid container"

Grid Line:
"garis horizontal / vertikal yg memisahkan grid mnjdi bbrp bagian dn ditandai dgn angka"

Grid Cell:
"perpotongan antara baris dn kolom di dlm grid"

Grid Area:
"kumpulan lbh dari 1 cell yg membentuk kotak"

Grid Track:
"ukuran / jarak antara 2 grid line, bisa horizontal atau vertikal"

Grid Gap:
"jarak antara cell"
